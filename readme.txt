Displays menu items on admin bar when user isn't logged in.

== Description ==
The plugin converts the new admin bar into menu bar when the user isnot logged in.


== Installation ==

1. Download the plugin.
2. Extract It.
3. Upload admin-bar-as-menu folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the Plugins section of WordPress admin area
3. You are set to go now.
